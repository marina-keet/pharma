# Gestion de Pharmacie

Ce projet est une application complète de gestion de pharmacie avec les fonctionnalités suivantes :
- Page de connexion
- Tableau de bord
- Gestion des clients
- Gestion des produits et du stock
- Point de vente (caisse)
- Rapports
- Gestion des factures
- Gestion des fournisseurs

## Stack technique
- Frontend : HTML, JavaScript, Tailwind CSS
- Backend : Node.js, Express
- Base de données : MySQL

## Installation
1. Cloner le dépôt
2. Installer les dépendances backend : `npm install`
3. Configurer la base de données MySQL dans le fichier `.env`
4. Lancer le serveur backend : `npm start`
5. Ouvrir le frontend dans un navigateur
